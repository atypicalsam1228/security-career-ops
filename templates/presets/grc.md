## Your Target Roles

| Archetype | Thematic axes | What they buy |

| **Cleared ISSM / Information System Security Manager** | RMF lifecycle, system authorization, POA&M management, continuous monitoring, eMASS, NIST 800-53, security plans | Someone who owns the full authorization posture of classified and unclassified systems |
| **ISSO / Security Control Assessor** | Control assessment, evidence collection, security documentation, vulnerability management, SCAP, ACAS | Someone who validates controls actually work and maintains system security posture day-to-day |

| **Security Auditor / Assessor (3PAO)** | Audit methodology, NIST 800-53A, penetration testing coordination, risk assessment, SAR writing, gap analysis | Someone who independently validates security posture and produces defensible assessment reports |
