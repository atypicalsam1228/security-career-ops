## Your Target Roles

| Archetype | Thematic axes | What they buy |
|-----------|---------------|---------------|
| **Cloud Security Engineer (AWS/Azure/GCP)** | IAM policy design, security services (GuardDuty, Defender, SCC), encryption, VPC security, logging, CloudTrail | Someone who secures cloud workloads with native tooling and defense-in-depth across multi-cloud environments |
| **Cloud Security Architect** | Landing zone design, multi-account strategy, zero trust, reference architectures, shared responsibility model | Someone who designs secure cloud foundations that scale with the organization |
| **DevSecOps / Pipeline Security Engineer** | CI/CD security gates, IaC scanning (tfsec, checkov), secrets management, container image signing, GitOps security | Someone who embeds security controls into deployment pipelines without slowing delivery velocity |
| **CSPM / Cloud Posture Engineer** | Prisma Cloud, Wiz, Orca, AWS Config, Azure Policy, drift detection, misconfiguration remediation, CIS benchmarks | Someone who maintains continuous visibility into cloud misconfigurations and drives posture improvement |
| **Container & Kubernetes Security Engineer** | Pod security standards, network policies, service mesh (Istio), runtime protection (Falco), image scanning, OPA/Gatekeeper | Someone who secures containerized workloads from build through runtime in orchestrated environments |
| **Identity & Access Management Engineer** | AWS IAM, Azure AD/Entra, RBAC/ABAC, federation (SAML/OIDC), least privilege, PAM, service account governance | Someone who designs and enforces identity-centric security as the new perimeter |
